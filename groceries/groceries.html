Sync Setup

Connect a Google Sheet so both of you see the same list in real time. Only one of you needs to do steps 1–4, then share the URL with the other.

  1. Go to sheets.google.com and create a new sheet called Grocery List.
  2. Click Extensions → Apps Script, delete any existing code, paste in the script below, and save.
  3. Click Deploy → New deployment. Type: Web app. Execute as: Me. Access: Anyone. Click Deploy.
  4. Copy the Web App URL it gives you and paste it below. Share that same URL with the other person — they paste it here too.
