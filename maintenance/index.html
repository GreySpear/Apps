Common home maintenance tasks with sensible schedules. Uncheck anything that doesn't apply — you can edit names and intervals afterwards.

Settings

Sync to a private Google Sheet so the log is backed up, shared across your devices — and so photos/receipts can be stored in your Drive. Everything else works offline.

Set-up steps & backend code
  1. Create a Google Sheet, open Extensions → Apps Script.
  2. Paste the maintenance/backend.gs from this repo, save.
  3. Deploy → New deployment → Web app. Execute as Me, access Anyone.
  4. Copy the Web App URL and paste it above.
