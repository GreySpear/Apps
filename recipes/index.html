    Settings

    Sync your recipes to a Google Sheet so they're backed up and shared across devices. Everything works offline too — text import always works; URL import needs the backend.

    Set-up steps & backend code
    1. Create a Google Sheet, open Extensions → Apps Script.
    2. Paste the backend.gs from this repo, save.
    3. Deploy → New deployment → Web app. Execute as Me, access Anyone.
    4. Copy the Web App URL and paste it above.

    The Recipes backend also fetches recipe web pages server-side for URL import (no CORS issues). See recipes/backend.gs & SETUP.md.
